Data gather prototypes

Respond to a child maintenance application - MVP

First iteration
Last updated
Feb 2024
Date created
Jan 2023
What it is
Includes:

- Section one / matching a user to a case and MCMC sign up

- Section two / asking a user for personal details

- Section three / optional equality questions section

- Section four / asking a user employment and income details

- Section five / asking a user about their children and arrangements

- Section six / payment types available for child maintenance

Changes from user research
whats new:

- Service name has been changed from "Complete a Child Maintenance Application" to "Respond to a Child Maintenance Application"

- Applicant name and Your name questions have been switched around

- MCMC sign up has been included

- Single name fields have been split into three fields

- Court order question has been added in as an eligibility filter

- Child benefit questions have been removed across the service

- Income questions have been merged into one question based on user feedback

- Parentage term has been changed to biological or legal parent

- A user is now dropped out of the service if they don't accept being the biological or legal parent

- Calculator link has been added to the end of the service based on user feedback

- DP onboarding and bank details have been aligned with Apply
